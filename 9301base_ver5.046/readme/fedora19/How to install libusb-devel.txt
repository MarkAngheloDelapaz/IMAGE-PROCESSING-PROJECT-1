カーネルのバージョンによってインストールするファイルが異なります。

画像処理開発課からリリースしたFedora19仮想マシンイメージには
複数のバージョンがありますが、カーネルのバージョンは下記２種類のどちらかになります。

3.11.9-200.fc19.i686.PAE
3.14.27-100.fc19.i686.PAE

それぞれインストールファイルが異なります。
ターミナルで下記コマンドを実行してカーネルのバージョンを確認してください。

uname -r

もしも上記以外のバージョンが表示された場合は、
画像処理開発課に御連絡下さい。




インストール方法

カーネルのバージョンにしたがって下記のどちらかの圧縮ファイルを任意のフォルダに解凍してください。
フォルダ内は解凍したファイルだけにしておいてください。

3.11.9-200.fc19.i686.PAEの場合		libusb-devel-0.1.5-2.zip
3.14.27-100.fc19.i686.PAEの場合		libusb-devel-0.1.5-3.zip

ターミナルで解凍先のフォルダをカレントディレクトリにして、
下記コマンドを実行してください。

sudo yum install *

[sudo] password for ipg:
と表示されるのでipgと入力してください。

--> Finished Dependency Resolution
と表示され数秒後にエラー表示されますが問題ありません。

Is this ok [y/d/N]: 
と表示されるのでyを入力してEnterを押してください。

以上です。

インストール時のログファイルを添付しているので参考にしてください。
